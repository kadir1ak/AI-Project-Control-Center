# Pricing Ideas

## Free

Amaç: bireysel deneme.

Kapsam:

- 1 proje.
- Manuel prompt generation.
- Sınırlı AI run.
- Basit Kanban.
- Temel PRD template.

## Pro

Amaç: solo founder ve bağımsız geliştirici.

Kapsam:

- 10 proje.
- AI PRD/backlog generation.
- Agent Runner.
- Prompt Engine.
- Terminal logs.
- GitHub entegrasyonu.

## Team

Amaç: küçük ekip ve ajans.

Kapsam:

- Çoklu kullanıcı.
- Approval workflow.
- Audit logs.
- Private workspaces.
- Gelişmiş agent profiles.
- Paylaşılan prompt templates.

## Enterprise

Amaç: kurumsal ekip.

Kapsam:

- On-prem veya private cloud opsiyonu.
- Custom model provider.
- SSO.
- Advanced permissions.
- Compliance logs.
- Dedicated support.

## İlk Fiyatlandırma Notu

MVP aşamasında fiyatlandırma yerine private beta ve pilot kullanıcı geri bildirimi öncelikli olmalıdır.
