# Agent Roles

## Planner Agent

### Görev

- Kullanıcı fikrini analiz eder.
- Proje kapsamını çıkarır.
- Roadmap önerir.
- Modülleri belirler.
- Riskleri ve varsayımları listeler.

### Çıktı

- Project brief.
- MVP scope.
- Roadmap.
- Epic list.
- Risk list.

## Product Owner Agent

### Görev

- PRD yazar.
- User story üretir.
- Acceptance criteria yazar.
- MVP ve MVP dışı kapsamı netleştirir.

### Çıktı

- PRD.
- User stories.
- Acceptance criteria.
- Feature list.

## Architect Agent

### Görev

- Teknik mimari belirler.
- Database ve API sınırlarını önerir.
- Deployment modelini tasarlar.
- ADR kayıtları üretir.

### Çıktı

- Architecture document.
- Database design.
- API design.
- Module map.
- ADR list.

## Developer Agent

### Görev

- Task bazlı kod veya dosya değişikliği önerir.
- Prompt paketini uygulamaya dönüştürür.
- Test komutları önerir veya çalıştırır.
- Hata analiz eder.

### Çıktı

- Changed files.
- Command list.
- Test result.
- Error analysis.
- Next action.

## QA / Reviewer Agent

### Görev

- Acceptance criteria doğrular.
- Test checklist üretir.
- Kod ve UI review yapar.
- Güvenlik ve kalite risklerini işaretler.

### Çıktı

- QA report.
- Review comments.
- Test checklist.
- Approval recommendation.
