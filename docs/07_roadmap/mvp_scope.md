# MVP Scope

## MVP Tanımı

İlk sürüm, tam otonom yazılım fabrikası değil; AI destekli yarı otonom planlama ve operasyon konsoludur.

## Yapılacaklar

- Proje oluşturma.
- Fikir girişi.
- AI ile PRD üretimi.
- PRD onayı.
- Epic ve task üretimi.
- Kanban board.
- Task detay sayfası.
- Ajan profilleri.
- Prompt paketi üretimi.
- Agent run kaydı.
- Approval Center.
- Temel log ekranı.
- Mermaid diagram taslağı.
- Wiki sayfası.

## Yapılmayacaklar

- Tam otonom production deploy.
- Enterprise role management.
- Marketplace.
- Çok karmaşık multi-agent framework.
- Kendi IDE editörünü sıfırdan yazmak.
- Çoklu tenant billing sistemi.
- Gelişmiş analytics.

## Başarı Kriteri

Kullanıcı bir proje fikrinden:

- PRD.
- Backlog.
- Mikro tasklar.
- Ajan görev paketleri.
- Approval gerektiren aksiyonlar.
- İzlenebilir run kayıtları.

üretebilmelidir.
