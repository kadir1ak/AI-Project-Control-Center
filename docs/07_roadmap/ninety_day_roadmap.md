# 90 Day Roadmap

## Ay 1: MVP

Çıktılar:

- Proje oluşturma.
- PRD üretme.
- Backlog oluşturma.
- Kanban board.
- Prompt Engine.
- Agent Runner.
- Basit log ekranı.
- Approval Center.

## Ay 2: Kod Üretim ve Git Entegrasyonu

Çıktılar:

- GitHub repo bağlama.
- Branch oluşturma.
- Dosya diff gösterme.
- PR hazırlama.
- Test komutları çalıştırma.
- Build logları.
- QA report üretme.
- Mermaid diagramlar.
- Wiki otomasyonu.

## Ay 3: Pilot Kullanım ve Ürünleştirme

Çıktılar:

- Gerçek bir SaaS projesinde pilot kullanım.
- Prompt template iyileştirme.
- Agent profile iyileştirme.
- Kullanıcı onboarding.
- Demo proje şablonları.
- Billing hazırlığı.
- Landing page.
- Private beta.

## 90 Gün Sonu Başarı Ölçütü

Bir kullanıcı, sistem üzerinden mini bir SaaS projesini planlayıp backlog'a bölebilmeli, ajan görev paketleri oluşturabilmeli, kontrollü run kayıtları alabilmeli ve GitHub entegrasyonuna hazırlayabilmelidir.
