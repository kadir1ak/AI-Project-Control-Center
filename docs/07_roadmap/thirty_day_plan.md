# 30 Day Plan

## Hafta 1: Temel Zemin

Amaç: manuel proje ve task yönetimi çalışan temel sistemi kurmak.

Yapılacaklar:

- Next.js UI kararını netleştir.
- .NET Web API temelini planla.
- PostgreSQL veri modelini kesinleştir.
- Projects, Epics, Tasks tablolarını hazırla.
- Basit Kanban UI tasarla.
- Task oluşturma ve düzenleme akışını tanımla.

Hafta sonu hedefi:

Kullanıcı manuel proje ve task oluşturabilmeli.

## Hafta 2: AI Planlama

Amaç: kullanıcı fikrinden PRD ve backlog üretmek.

Yapılacaklar:

- AI provider abstraction tasarla.
- Claude/OpenAI bağlantı planını hazırla.
- Ideation ekranı akışını netleştir.
- PRD generation servisini tasarla.
- PRD'den epic/task üretim promptlarını hazırla.

Hafta sonu hedefi:

Kullanıcı fikrinden PRD ve ilk backlog üretilebilmeli.

## Hafta 3: Agent Runner ve Prompt Engine

Amaç: task bazlı ajan çalışma sistemini kurmak.

Yapılacaklar:

- AgentProfiles modelini netleştir.
- PromptTemplates modelini netleştir.
- ContextDocuments modelini netleştir.
- PromptEngineService davranışını tanımla.
- AgentRun ve AgentRunSteps ekranını tasarla.

Hafta sonu hedefi:

Bir task için güçlü ve tekrar kullanılabilir prompt paketi üretilebilmeli.

## Hafta 4: Approval ve Worker Hazırlığı

Amaç: kontrollü terminal ve onay sisteminin temelini kurmak.

Yapılacaklar:

- Command policy modelini tasarla.
- Approval tablosunu ve ekranını netleştir.
- Worker runtime MVP kararını ver.
- Terminal log stream gereksinimlerini tanımla.
- File diff ekranının minimum kapsamını belirle.

Hafta sonu hedefi:

Riskli işlemler approval'a düşecek şekilde tasarlanmış olmalı.
