# Demo Scenario

## Demo Proje

Mini CRM SaaS.

## Kapsam

- Authentication.
- Customer management.
- Contact management.
- Simple dashboard.
- PostgreSQL database.
- .NET Web API backend.
- Next.js frontend.
- Docker compose.
- README.
- Basic tests.

## Demo Akışı

1. Kullanıcı "Mini CRM SaaS yapmak istiyorum" der.
2. Sistem ürün fikrini yapılandırır.
3. Sistem PRD üretir.
4. Kullanıcı PRD'yi onaylar.
5. Sistem mimari özet üretir.
6. Sistem database ve API taslağı üretir.
7. Sistem backlog oluşturur.
8. Kullanıcı planı onaylar.
9. Sistem auth modülü için task paketleri üretir.
10. Agent Runner ilk developer run kaydını oluşturur.
11. Worker veya manuel araç için prompt paketi hazırlanır.
12. Test ve review sonucu sisteme kaydedilir.

## Demo Başarı Mesajı

Bu demo, ürünün "fikirden uygulanabilir geliştirme operasyonuna" geçişi nasıl sistematik hale getirdiğini göstermelidir.
